# Employee Data Analysis Tool

## 📌 Project Overview

The Employee Data Analysis Tool is a Python-based application developed using **Pandas** and **Matplotlib**. It reads employee information from a CSV file, performs data analysis, and displays useful insights along with graphical visualizations.

---

## ✨ Features

- Read employee data from a CSV file
- Display employee records
- Calculate total number of employees
- Calculate average salary
- Find highest salary employee
- Find lowest salary employee
- Department-wise average salary analysis
- Search employee by name
- Filter employees by department
- Export data to a new CSV file
- Generate Bar Chart
- Generate Pie Chart
- Generate Line Chart

---

## 🛠️ Technologies Used

- Python
- Pandas
- Matplotlib
